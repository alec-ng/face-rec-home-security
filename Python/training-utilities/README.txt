==================
Contents
==================

Contains scripts that generates a new user in the system and 
trains the face for face recognition.
create_user.py does this.

==================
Subdirectories
==================

** ->  Directories exist locally. Listed here because our scripts reference
	   these directories.

|------ **Image-Raw ->
|		contains subdirs (named with unique subject IDs) of unedited images
|
|------ **Image-Align ->
|		contains subdirs of raw images that have been aligned
|
|------ **Image-Process ->
|		contains subdirs of final procesed images
